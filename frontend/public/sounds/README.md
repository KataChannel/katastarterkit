# Notification Sound

This folder contains sound files for the Support Chat widget.

## Required Files

- `notification.mp3` - Sound played when a new message is received

## Recommended Sound

You can use any short notification sound. Recommended specs:
- Duration: 0.5 - 1 second
- Format: MP3
- Size: < 50KB

## Free Sound Resources

- [Freesound.org](https://freesound.org/search/?q=notification)
- [Mixkit](https://mixkit.co/free-sound-effects/notification/)
- [Zapsplat](https://www.zapsplat.com/sound-effect-categories/notification-sounds/)

## Example Sound

A simple notification sound can be generated using these specifications:
- Tone: Single chime or bell
- Frequency: ~800Hz - 1200Hz
- Duration: 300ms - 500ms
- Fade out: 200ms
