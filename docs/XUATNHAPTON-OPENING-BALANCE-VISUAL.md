## ðŸ§® Formula Reference

### Opening Balance
```
openingQuantity[day] = closingQuantity[day - 1]
openingAmount[day] = closingAmount[day - 1]

Special case (first day):
openingQuantity[first] = 0
openingAmount[first] = 0
```

### Weighted Average Cost Price
```
WAVG = (openingAmount + importAmount) / (openingQuantity + importQuantity)

If denominator = 0: WAVG = 0
```

### Closing Balance
```
closingQuantity = openingQuantity + importQuantity - exportQuantity
closingAmount = openingAmount + importAmount - exportAmount

Where:
  exportAmount = exportQuantity Ã— WAVG cost price
```
