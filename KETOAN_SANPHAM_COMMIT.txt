## New Features

### 1. Product Management Page (/ketoan/sanpham)
- Display products from ext_sanphamhoadon table
- Show fields: ten, ten2, ma, dvt, dgia
- Search functionality across product names and codes
- Pagination support (50 items per page)
- Real-time statistics dashboard

### 2. Product Normalization with Fuzzy Matching
- Integration with pg_trgm fuzzy matching system
- Normalize product names using similarity algorithm
- Preview mode (dry-run) before actual update
- Configurable batch sizes (10, 100, 1000, or all)
- Interactive normalization modal

### 3. Statistics Dashboard
- Total products count
- Normalized products count (ten2 not null)
- Pending normalization count (ten2 null)
- Visual status badges for each product

## Technical Details

### Database Schema
```typescript
model ext_sanphamhoadon {
  id: String
  ten: String  // Original name
  ten2: String // Normalized name ⭐
  ma: String   // Product code
  dvt: String  // Unit
  dgia: Decimal // Price
}
```

### Fuzzy Matching
- Uses PostgreSQL pg_trgm extension
- GIN index on ten field for fast similarity search
- Configurable similarity threshold (default 0.6)
- Groups similar products automatically

### API Flow
```
Frontend → Next.js API Route → Backend Controller → updateten2.js Script
```
